Idea Box

(GitHub)

Idea box is an application, created by colleague Adam Rice and I, that allows the user to enter in a title and body of an "idea", which then populate in an area below the input box. The user can assign one of three "qualities" to an idea ("genius", "plausible", or "swill"), delete an idea, and search for text within any ideas' titles and bodies. All of this persists after page reload thanks to local storage. For more details, see the project page.
